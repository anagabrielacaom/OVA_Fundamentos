¿Qué es un algoritmo?

Variable y Tipo de Datos

Las variables almacenan datos para ser procesadas. Una variable le es dado un nombre o identificador.

  • Las variables almacenan datos para ser procesadas.
  • A una variable le es dado un nombre o identificador.
  • El nombre identifica de manera única cada variable, asignando un valor a la variable y tomando el valor almacenado.
  • Se puede declarar una variable de un tipo y asignarle un valor.
  • Una variable debe ser declarada antes de poder ser utilizada.

Tipos de datos primitivos

El tipo de dato String se utiliza para textos. Las cadenas de texto están delimitadas dentro de comillas dobles.

char corresponde a carácter y almacena un solo carácter.

Se puede utilizar una lista separada por comas para declarar más de una variable del tipo especificado.

Palabras reservadas

Ciertas palabras están reservadas para uso interno por Java y no se pueden utilizar como nombres de variables.

Las palabras reservadas cont y goto son reservadas pero no se utilizan.

Nota. Además de estas palabras reservadas, Java se reserva false,null y true como valores definidos en el lenguaje
